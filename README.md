<p>1) git init <br>
2) git add .  <br>
3) git commit -m 'write what you committed'  <br>
4) git branch -M main  <br>
5) git remote add origin <copy the repo URL here> example. https://github.com/trevorm10/The-Calculator.git  <br>
6) git push -u origin main </p>
